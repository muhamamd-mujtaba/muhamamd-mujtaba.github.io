# Muhammad Mujtaba Portfolio

This portfolio is a static website built with HTML, CSS, and JavaScript.

## Main Pages

- `index.html`
- `about.html`
- `contact.html`
- `blog.html`

## Blog Posts

Each article is stored as its own HTML file inside the `posts` folder.

To add a new post later:

1. Create a new file such as `posts/article-11.html`.
2. Follow the same design pattern as the existing article pages.
3. Add its link card inside `blog.html`.

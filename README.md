# Muhammad Mujtaba Portfolio

This portfolio now uses Jekyll so the shared layout stays reusable while the public site still builds into static pages.

## Main Pages

- `index.html`
- `about.html`
- `contact.html`
- `blog.html`

## Blog Posts

Each article is managed through the `_posts` folder.

To add a new weekly post later:

1. Create a new file such as `_posts/2026-04-11-article-11-new-story.md`.
2. Add front matter for `title`, `summary`, `article_number`, `tags`, and `permalink`.
3. If you have a custom image, add `image: /assets/your-image-name.svg` or `image: /assets/your-image-name.jpg`.
4. If you do not add an image, the site will use the default article visual automatically.
5. Write the article content below the front matter.
6. Push to GitHub Pages and the blog page will update automatically.

# Muhammad Mujtaba Portfolio

This portfolio now uses Jekyll so the shared layout stays reusable while the public site still builds into static pages.

## Main Pages

- `index.html`
- `about.html`
- `contact.html`
- `blog.html`

## Blog Posts

Each article is managed through the `_posts` folder.

To add a new post later:

1. Create a new file such as `_posts/2026-04-04-article-11-new-story.md`.
2. Add front matter for title, summary, article number, tags, and permalink.
3. Write the article content below the front matter.
4. Push to GitHub Pages and the blog page will update automatically.
